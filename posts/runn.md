RUNN = RNN + run.

It is a side-scrolling game based on a musical machine learning algorithm that can generate melodies. The player has to finish the level to hear the full song.

The game uses Tone.js and MusicVAE. You run and jump while staying in the purple band to keep the musical vibe alive.

- [Learn more about the model](https://magenta.tensorflow.org/music-vae)
